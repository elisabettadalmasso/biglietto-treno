# Biglietto Treno

Calcolatore del prezzo di un biglietto ferroviario, sviluppato in React + TypeScript + Tailwind CSS.

## Funzionalità

- Calcolo del prezzo in base ai chilometri percorsi (0.21€/km)
- Sconto del 20% per i passeggeri minorenni
- Sconto del 40% per i passeggeri over 65
- Prezzo finale arrotondato a due decimali

## Stack

- React
- TypeScript
- Tailwind CSS
- Vite
